# voice-call-twilio-notification

A personal alert system that monitors Discord channels and Telegram channels for signals (memecoin token addresses) and calls your phone via Twilio.

## Features

- **Discord bot** — monitors channels, calls you when a message arrives
- **Telegram tracker** — monitors private channels, detects EVM + Solana token addresses and calls you
- **Cooldown** — configurable per source to prevent spam calls
- **PM2** — process manager for always-on background running

---

## Quick Start

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment

```bash
cp .env.example .env
```

Fill in `.env` — see [Environment Variables](#environment-variables) below.

### 3. Telegram auth (one-time)

```bash
npm run telegram:auth
```

Follow the prompts (phone number + OTP). Copy the printed `TELEGRAM_SESSION` value into `.env`.

### 4. Run

```bash
# Development
npm run dev

# Production (PM2)
npm run pm2:start
```

## Environment Variables

```env
# Server
PORT=3000
NODE_ENV=development
LOG_LEVEL=info

# Twilio
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE_NUMBER=       # your Twilio number (E.164 format)
CALL_TO_NUMBER=            # your personal number to receive calls

# Discord Bot
DISCORD_BOT_TOKEN=
DISCORD_COOLDOWN_MS=300000
DISCORD_CHANNEL_ID=        # optional: blank = all channels

# Telegram
TELEGRAM_API_ID=           # from my.telegram.org
TELEGRAM_API_HASH=         # from my.telegram.org
TELEGRAM_SESSION=          # generated by: npm run telegram:auth
TELEGRAM_CHANNEL_ID=       # optional: blank = all channels
TELEGRAM_COOLDOWN_MS=300000
```

## Discord Setup

1. Create a bot at https://discord.com/developers/applications
2. Under **Bot** → enable **Message Content Intent**
3. Copy the bot token → `DISCORD_BOT_TOKEN`
4. Invite the bot to your server with `View Channels` + `Read Messages` permissions

---

## Telegram Setup

1. Go to https://my.telegram.org → **API development tools**
2. Create an app → copy `api_id` and `api_hash`
3. Run `npm run telegram:auth` to generate your session string
4. Set `TELEGRAM_CHANNEL_ID` to the channel ID or username

**Token address detection:**
- EVM chains (ETH, BSC, etc.): `0x` + 40 hex characters
- Solana: base58 string, 43–44 characters

## Voice Call Provider

This project uses **Twilio** by default. The free trial only allows calls to **verified phone numbers** — meaning you can only call numbers you've manually added and confirmed in the Twilio console.

### Workaround (Free)

Twilio allows multiple free trial accounts. You can create a new account to get a fresh $15 credit with a different verified number. Useful for testing with different numbers.

### Recommended Upgrade: Switch to Telnyx

To remove all restrictions and call any number, switch to a provider without trial limitations:

| Provider | Price/min | Trial Restrictions |
|---|---|---|
| **Telnyx** | ~$0.002 | None — $10 free credit |
| **Plivo** | ~$0.004 | None — $5 free credit |
| **Vonage** | ~$0.016 | None — $2 free credit |
| **Twilio** | ~$0.013 | Verified numbers only on trial |

**Telnyx** is the recommended alternative — cheapest rate, no restrictions, and the API is nearly identical to Twilio so the code change is minimal (swap credentials + SDK).
